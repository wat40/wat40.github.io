Step 1: Invite NexusBot

Invite NexusBot

Manual Setup

  1. Go to the Discord Developer Portal
  2. Select your application or create a new one
  3. Go to the "OAuth2" → "URL Generator" section
  4. Select "bot" and "applications.commands" scopes
  5. Choose the required permissions (see recommended permissions)
  6. Copy the generated URL and open it in your browser
  7. Select your server and authorize the bot

Step 2: Basic Configuration

Configure essential settings to get started.

Set Command Prefix

Change the default command prefix from - to something else:

-prefix !

Set Moderation Channel

Designate a channel for moderation logs:

-modchannel #mod-logs

Configure Welcome System

Set up welcome messages for new members:

-welcome #general Welcome to the server, {user}!

Set Auto Role

Automatically assign a role to new members:

-autorole @Member
